# Sketch 004: Projecten fototreatment

## Design Question
The real install photos (red brick, warm oak, dark attics, bright bedrooms) read
heavier and noisier than the light `#f1fbfe` design. Which uniform *presentation
treatment* makes them feel curated instead of pasted on — without hiding that
they're real, unstaged proof?

## How to View
open ".planning/sketches/004-projecten-photo-treatment/index.html"
(photos load from `public/images/projecten/` via relative paths)

## Variants
- **A: Kalm kleurenpalet** — one shared cool grade (desaturate + lift + petrol color-blend); full color stays, but calmed toward the palette. Cheapest, most honest.
- **B: Galerij-passe-partout** — white mat frames + cool shadows + captions; the frame unifies, photos keep natural color. Most "premium print" feel.
- **C: Duotone → kleur** — petrol monochrome at rest, true color on hover/touch; maximum rest + one tasteful interaction (house DNA), but hides color until interaction (mobile = tap).
- **D: Atmosferisch ingebed** — feathered edges melt photos into the tonal card + grade from A + technical brand·month label; most atmospheric/engineered.

## What to Look For
- Which treatment tames the **red brick** and **warm oak** shots best?
- Does the treatment still feel like *real proof* (not over-designed)?
- Hover each photo: the signature top-accent reveal is on all variants.
- C on mobile: color only appears on tap — acceptable trade-off?
- Toolbar bottom-right: try 📱 375px.
