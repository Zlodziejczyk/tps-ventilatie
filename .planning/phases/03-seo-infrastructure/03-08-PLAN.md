# 03-08 — Owner runbook + SEO assertions + phase verification

## Objective

Ship the credential-bound owner tasks as a written runbook (D-07: GBP, GSC, Vercel enable,
AI-crawler opt-out, geo confirm — SEO-07/SEO-09), add the project-idiomatic `scripts/assert-seo.ts`
that locks the indexing policy + JSON-LD invariants, and run the whole-phase gate: green static
export with the built `out/` proving sitemap/robots/JSON-LD/canonical/noindex all resolve and the
stale brand is gone.

Purpose: the phase's "is it actually wired?" gate + the single doc the owner works from to finish
the external steps. Output: `docs/seo-owner-runbook.md`, `scripts/assert-seo.ts`, a green build.

## must_haves

- `docs/seo-owner-runbook.md` lists all 5 owner task groups (D-07): **GBP** (name "TPS
  klimaattechniek"; primary category `Airconditioningsbedrijf` flagged for owner confirmation;
  secondaries `Verwarmingsinstallateur`/`Ventilatiebedrijf`/`HVAC-aannemer`; service-area from
  `SITE.serviceAreas`; verify maps pin/location); **GSC** (set `NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION`,
  confirm verification, submit `https://tpsventilatie.nl/sitemap.xml`, optional DNS-TXT upgrade);
  **Vercel** (enable Analytics + Speed Insights); **AI-crawler opt-out** block list (documented, not
  enforced); **geo** (confirm the real business lat/lng for JSON-LD `geo` + the GBP pin — `SITE.geo`
  is the owner-verify-pending placeholder, A-1) + re-confirm the `www→apex` 301 pre-launch.
- `scripts/assert-seo.ts` (Node `assert/strict`, run via `npx tsx`, mirrors `assert-registry.ts`)
  passes: `isIndexable` yields exactly the 4 static content pages; privacy-beleid + hub + all
  pillars/services excluded; `sitemapEntries()` length 4 and every url absolute-apex; `CANONICAL_ORIGIN`
  === `"https://tpsventilatie.nl"`; `businessJsonLd()` is `@type "HVACBusiness"` with a `/#business`
  `@id` and NO `aggregateRating`/`review`; `faqJsonLd(emptyNode)` === null.
- `npm run build` exits 0 (prebuild taxonomy gate + static export) and `out/` proves the wiring (greps below).
